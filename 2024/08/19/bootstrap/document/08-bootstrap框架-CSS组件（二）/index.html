bootstrap框架-CSS组件（二）

徽章

定义徽章

徽章（Badge）是一种小型的标签或标记，用于显示额外的信息或数字统计数据，如通知数量、状态或分类标签。徽章通常附加在其他元素（如按钮、导航栏、文本）旁边，以提供额外的上下文信息。Bootstrap 提供了便捷的方式来创建和样式化徽章。

按钮徽章

设置颜色

Bootstrap 提供了一组预定义的背景颜色类，你可以应用于徽章以区分不同的状态或类别。这些类包括 bg-primary，bg-secondary，bg-success，bg-danger，bg-warning，bg-info，bg-light，和 bg-dark。

椭圆形徽章

通过将 rounded-pill 类添加到徽章元素上，可以将徽章设计为椭圆形，这通常用于柔化外观或更符合某些设计风格。

链接徽章

有时，你可能希望徽章本身是可点击的链接。在这种情况下，你可以将徽章包裹在 <a> 标签内，使其具有链接功能。Bootstrap 会自动处理样式，使其看起来像一个按钮或链接，同时仍然保留徽章的外观。

警告框

警告框（Alert）是一种用于向用户显示重要通知、警告或其他信息的界面组件。在Bootstrap中，警告框可以通过简单的HTML和预定义的类来实现，并且可以根据需要自定义样式和行为。

定义警告框

Bootstrap 提供了多种预定义的警告框样式，适用于不同类型的消息。可以通过添加 alert 类和相应的颜色类（如 alert-primary，alert-secondary，alert-success，alert-danger，alert-warning，alert-info，alert-light，alert-dark）来定义警告框。

可关闭的警告框

警告框可以设置为可关闭的，让用户可以通过点击关闭按钮来消除警告框。通过在警告框内添加一个关闭按钮，并结合 alert-dismissible 类和 fade show 类来实现这一功能。

含有其他元素的警告框

你可以在警告框中包含标题、段落、链接等其他元素，以丰富警告框的内容。通过在警告框内添加这些元素，可以使警告信息更具层次和清晰度。

自定义警告框

你可以通过自定义 CSS 来更改警告框的外观和行为。例如，可以更改警告框的背景颜色、文本颜色、边框样式等。

警告框的用途

警告框可以用于以下几种场景：

  • 信息通知：提示用户重要信息或系统更新。
  • 成功提示：确认用户操作已成功完成。
  • 警告信息：提示用户潜在风险或需要注意的事项。
  • 错误消息：告知用户操作失败或系统错误。

媒体对象

媒体对象（Media Object）是Bootstrap中一种非常灵活且强大的布局模式，主要用于显示一系列相关的媒体内容，如图片、视频、文本等。媒体对象通过简单的HTML结构，可以轻松实现复杂的媒体布局。

媒体版式

媒体对象的基本结构包括一个媒体（如图像或视频）和相应的文本内容。Bootstrap提供了 media 类，用于创建媒体对象版式。

媒体嵌套

媒体对象可以嵌套在一起，以实现更复杂的布局。例如，一个媒体对象可以包含另一个媒体对象，以展示层级结构或多级信息。

对齐方式

媒体对象中的图像或视频可以根据需要进行对齐。可以使用 align-self-start、align-self-center 和 align-self-end 类来分别实现顶部对齐、居中对齐或底部对齐。

排列顺序

在媒体对象中，可以通过调整HTML结构，改变图像和文本的排列顺序。例如，可以将图像放在右边而非左边。

媒体列表

当需要展示一系列相似的媒体对象时，可以将它们放在一个列表中。Bootstrap 提供了 list-unstyled 类，可以创建一个无序的媒体对象列表。

进度条

进度条（Progress Bar）是一种常用的界面组件，用于显示任务的进度或完成情况。它能够帮助用户直观地了解某个操作的执行情况，常见于加载状态、上传进度、下载进度等场景。

定义进度条

Bootstrap 提供了基本的进度条组件，可以通过简单的 HTML 结构来实现。基本的进度条包括一个外部容器和一个内部进度条，进度条的宽度可以根据完成比例进行调整。

Bootstrap 提供了一些预定义的进度条样式，如背景颜色和条纹样式，你可以使用这些样式来设计不同的进度条外观。也可以通过自定义 CSS 来进一步调整进度条的样式。

导航栏

导航栏（Navbar）是网站中一个重要的组件，用于提供网站或应用程序的主要导航链接。Bootstrap 提供了一个功能强大的导航栏组件，可以通过简单的 HTML 和 CSS 实现各种样式和功能。

定义导航栏

导航栏的基本结构包括一个容器、一个品牌（通常是网站名称或 logo）、以及导航链接。Bootstrap 的导航栏组件使用 navbar 类进行定义，并提供了多种布局选项。

导航栏可以固定在页面顶部、底部或设置为响应式导航栏，使其在滚动时仍保持可见。Bootstrap 提供了 fixed-top、fixed-bottom 和 sticky-top 类来控制导航栏的位置。

扩展导航栏内容
